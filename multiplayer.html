Vietcong Server List Fix

GameSpy has discontinued its support for Vietcong, which means no servers will appear in the in-game server list. However, you can still host servers and connect directly to them.

If you use VcComHub, this fix is in common.

How to fix?

  1. Copy to desktop file C:\WINDOWS\system32\drivers\etc\hosts
  2. Add 46.28.109.117 master.gamespy.com
  3. Save file and paste it back

For Server Hosts

If you host a server, you must also apply this change; otherwise, your server won't appear in the list. If desired, you can still use the old Qtracker IP:

65.112.87.186 master.gamespy.com

Note on Qtracker

Qtracker's master server is still functional but its future is uncertain. Vietcong1.eu provides an open-source alternative, and multiple master servers can synchronize lists to form a network.
