Maps

For playing multiplayer you need maps.

How to download?

  1. Download maps (zip, one map, etc.)
  2. Every map has two files (<map-name>.dat, <map-name>.cbf), you need both of them
  3. Close vietcong
  4. Paste this files to vietcong/maps folder
  5. Run vietcong again

Important: in maps folder can be only 1000 files = 500 maps
